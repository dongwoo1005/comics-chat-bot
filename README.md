# Battlecomics 페북 메신저 봇

How to Install
--------------
```
$ npm install
```

How to Run the Server for Development
---------------------
Nodemon reloads the server automatically.
```
$ npm dev 
```

How to Run the Server for Hosting
--------------------
Babel compiles ES6 source files to distribute version
```
$ npm start
```
